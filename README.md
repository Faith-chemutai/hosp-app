## Project SetUp

- Run and install our dependency gems in the Gemfile in the terminal:

```bash
bundle install
```

- Run the migrations:

```bash
rails db:migrate
```

- Seed the database data:

```bash
rails db:seed
```

- Run the server:

```bash
rails s
```

## React Set-up

- To set up the React frontend:

```bash
npm install --prefix client
```

- Run the server:

```bash
npm start --prefix client
```
